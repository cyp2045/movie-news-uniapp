# -
基于uniapp+Vue3开发的影视新闻一体化App【影闻汇】，整合全网新闻聚合API与影视资源接口，一套代码兼容小程序/H5/Android/iOS，实现影视、资讯一站式浏览，自带统一缓存优化与跨端适配逻辑。

### Git 上传注意事项
本项目已配置 `.gitignore` 文件，以下目录不会上传至远程仓库：
- `uni_modules/` uni-app组件依赖
- `node_modules/` npm依赖包
- `unpackage/` 编译打包缓存
- `.hbuilderx/` HBuilderX配置缓存
- `*.log` 日志文件

克隆项目后，使用 HBuilderX 重新下载对应 uni_modules 组件即可正常运行。
